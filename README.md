# YUIviewer
 将YUI格式的视频文件经过数字处理后变为rgb像素点绘制在画面上。  
 本人第一个使用MVC设计模式的java作品，如果有对此设计模式上更优化的方案，请告知我。

# 使用方式
 点击YUIviewer-Jar下的.bat文件  
 未清楚为何不能单独点击jar来运行,希望有大佬解答。
